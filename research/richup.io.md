# Research: Richup.io

## What is Richup.io?
A free, browser-based online multiplayer board game serving as a Monopoly alternative. No sign-up or download required — players visit the website and play instantly.

## Core Gameplay
- All players start with **$1500**
- Players take turns **rolling dice** to move around the board
- **Doubles** grant an extra turn
- Goal: **bankrupt all other players** by acquiring properties and collecting rent

## Property System
- Land on unowned properties to purchase them
- Owning a **complete color set** allows building **houses and hotels**
- Rent increases exponentially with improvements
- **Double rent** applies when owning a full set

## Game Modes
- **Friends** — private rooms
- **Strangers** — public rooms
- **Bots** — AI opponents

## Configurable Rules
- Starting cash options ($1,500–$3,000)
- **Auction system** for unpurchased properties
- **Mortgage** feature to sell properties
- **Vacation cash** (tax income distribution)
- Option to disable rent collection while in jail

## Features
1. Real-time multiplayer with multiple players
2. Private rooms for friends
3. Bot players when humans aren't available
4. Full property trading system between players
5. House/hotel construction on complete property sets
6. In-game store for cosmetic upgrades
7. Tournaments (first tournament November 2022)
8. Active Discord community
9. Blog with strategy guides and game updates

## UX Strengths
1. **Clean visual interface** — well-designed, clean UI
2. **No installation required** — instant browser play
3. **Flexible matchmaking** — play with friends, strangers, or bots
4. **Configurable rules** — game hosts can customize (starting cash, auctions, mortgage, etc.)
5. **Accessible on desktop** — optimized for PC/laptop
6. **Social features** — friend system, blocking, Discord community
7. **Tutorial/how-to-play** — built-in guidance for new players
8. **Consistent updates** — active blog, tournaments, new features
